# Professor Huping Ling website

This site is built with Jekyll and hosted by GitHub Pages.

## Content structure

- `_posts/` — news and writing
- `_data/books.yml` — books
- `_data/talks.yml` — recordings and appearances
- `_data/awards.yml` — honors
- `_layouts/` and `_includes/` — shared page structure
- `assets/css/main.css` — visual system
- `admin/` — browser-based content editor

## Local preview

```sh
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000`.

See `ADMIN_SETUP.md` for content-editor sign-in instructions.
